### 3.1 MarketShare 플랫폼 현황 분석

MarketShare는 기존 플랫폼들과 근본적으로 다른 "**통합 마켓 + 분양몰**" 하이브리드 모델이다. 현재 시스템에 이미 다음 기능이 구현되어 있다:

- **5가지 테마 시스템**: 기본, 쇼핑몰, 기업/브랜드, 음식점/배달, 서비스업
- **분양 신청 프로세스**: 테마 선택 → 기본 정보 입력 → 신청 → 관리자 심사
- **몰 레벨 시스템**: 1~5단계 (MallLevel)
- **PG 결제 인증 방식**: 플랫폼 통합 / 개별 / 선택적 (PGPaymentAuth)
- **정산 시스템**: 수수료 산정, 추천 수수료, 판매 수수료 분리 관리
- **폐쇄몰/승인 기반 몰**: isClosedMall, requireMemberApproval 지원
- **통합 결제**: PortOne(아임포트) 기반 다중 PG 연동

### 3.4 정산 구조

| 항목 | 내용 |

| **정산 방식** | 총 매출 - 판매수수료 - PG수수료 - 추천수수료 = 정산금액 |

| **정산 상세** | 관리자 페이지에서 주문별 수수료 내역 실시간 조회 가능 |

#### 정산 계산 예시 (Business 플랜)

```
상품 판매가: 100,000원
배송비: 3,000원 (수수료 계산 제외)

PG 수수료 (카드 2.5%): 100,000 x 2.5% = 2,500원
판매 수수료 (1.5%): 100,000 x 1.5% = 1,500원
추천 수수료 (해당 시): 100,000 x 0.5% = 500원

정산 금액: 100,000 - 2,500 - 1,500 - 500 + 3,000(배송비) = 98,500원
```

#### 차별화 1: 메인 마켓 자동 노출

**기존 플랫폼의 한계**: 카페24, 메이크샵, 고도몰 등은 독립 쇼핑몰만 제공하여 트래픽을 판매자가 스스로 확보해야 한다. 식스샵, 아임웹도 마찬가지로 개별 몰만 운영되며 통합 마켓은 없다.

**MarketShare의 해결**: 분양몰 개설 즉시 MarketShare 메인 마켓에 상품이 자동 노출된다.

| 노출 영역 | Free | Starter | Business | Enterprise |
|----------|------|---------|----------|------------|
| 카테고리별 상품 목록 | O | O | O | O |
| 통합 검색 결과 | O | O | O | O |
| 메인 추천 상품 | X | O | O (우선) | O (최상위) |
| 기획전/프로모션 | X | X | O | O |
| 전용 배너 영역 | X | X | X | O |
| 브랜드관 입점 | X | X | O | O |

> 기존 플랫폼에서 쇼핑몰을 개설하면 "텅 빈 매장"에서 시작하지만, MarketShare에서는 "유동인구가 있는 백화점에 입점"하는 효과를 제공한다.

#### 차별화 2: 5초 만에 개설

**기존 플랫폼의 문제**:
- SB플러스/투비웹: 견적 협의 → 개발 → 설치까지 수일~수주 소요
- 카페24/메이크샵/고도몰: 가입 → 설정 → 디자인까지 수시간~수일 소요
- 식스샵: "6분 만에" 슬로건

**MarketShare의 해결**:
1. 분양 신청 페이지에서 테마 선택 (1초)
2. 기본 정보 입력 (4초 - 소셜 로그인 시)
3. 즉시 개설 완료 (Free 플랜)

Free 플랜은 본인인증만으로 즉시 개설되며, 테마가 자동 적용되어 별도의 디자인 작업이 불필요하다. 플랫폼 상품 DB에서 상품을 가져와 즉시 판매를 시작할 수도 있다.

- 개인사업자 초기 PG 심사 거절 리스크

**MarketShare의 해결**:

```
[플랫폼 통합 PG 방식]

고객 결제 → MarketShare PG → MarketShare 정산 → 몰 운영자 계좌 입금

장점:

- 신청 즉시 결제 수락 가능 (심사 대기 없음)

- 초보 판매자도 결제 인프라 걱정 없음
```

```
[개별 PG 방식 - Business/Enterprise]

장점:
```

## 4부: 결론 및 핵심 요약

### MarketShare가 경쟁에서 이기는 3가지 핵심

1. **"개설 즉시 고객이 있다"**: 메인 마켓 자동 노출로, 분양몰을 개설하는 순간부터 MarketShare의 기존 트래픽을 활용할 수 있다. 카페24, 메이크샵 등 기존 플랫폼에서는 불가능한 가치다.

2. **"5초 개설, 0원 시작"**: Free 플랜으로 사업자 없이도 즉시 개설 가능하며, 플랫폼 통합 PG로 결제 인프라 고민 없이 바로 판매를 시작할 수 있다.

3. **"성장에 따라 확장"**: Free → Starter → Business → Enterprise로 비즈니스 규모에 맞게 업그레이드할 수 있으며, Enterprise에서는 하위 분양까지 가능한 확장성을 제공한다.
