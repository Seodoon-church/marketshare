## 해결한 기술 이슈

1. **Tailwind v4 CSS 변수 순환 참조** - `rgb(var(...))` → 직접 hex 값 사용
2. **CSS @import 순서 오류** - Pretendard 폰트를 `<link>` 태그로 이동
3. **--content-max-width 미적용** - `:root`에 기본값 1280px 추가
4. **Firebase Admin 빌드 크래시** - Lazy Proxy 패턴으로 지연 초기화
5. **Zod v4 문법 차이** - `errorMap` → `error` 속성으로 변경
6. **Tiptap BubbleMenu 미지원** - 해당 버전에서 제거
7. **useAuth 타입 불일치** - UserCredential vs User 반환 타입 정리
