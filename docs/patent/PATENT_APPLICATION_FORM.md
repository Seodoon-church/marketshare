## 온라인 출원 단계별 가이드

### 1단계: 특허로 접속 및 로그인
1. https://www.patent.go.kr 접속
2. 공인인증서 또는 간편인증(카카오/네이버 등) 로그인
3. "출원" → "특허출원" 선택

### 2단계: 출원서 작성
1. **발명의 명칭** 입력: "프랜차이즈 몰 네트워크 기반 다중 플랫폼 라이브 커머스 통합 관리 시스템 및 방법"
2. **출원인/발명자 정보** 입력
3. **국제특허분류(IPC)** 선택: G06Q 30/06
4. **출원 구분**: 일반출원 선택
5. **심사청구 여부**: 출원과 동시에 심사 청구 (권장)

### 3단계: 명세서 첨부
1. **특허명세서**: PATENT_SPECIFICATION.md 내용을 HWP 또는 PDF로 변환 후 업로드
2. **청구범위**: PATENT_CLAIMS.md의 12개 청구항 입력
3. **요약서**: 명세서 9장의 395자 요약문 입력
4. **도면**: 7개 다이어그램을 이미지(PNG/JPG)로 변환 후 업로드

### 4단계: 서류 변환 가이드
- Markdown → HWP 변환: 무료 온라인 변환기 활용 또는 한글 문서에 직접 붙여넣기
- 도면: 명세서의 ASCII 다이어그램을 깔끔한 블록 다이어그램으로 작성
  - 무료 도구: draw.io (diagrams.net), Figma
  - 특허 도면 규격: A4 세로, 여백 2cm, 흑백

### 5단계: 수수료 납부
1. 출원료 + 심사청구료 납부
2. 감면 대상 시 감면율 적용
3. 온라인 카드결제 또는 계좌이체

### 6단계: 출원 완료
1. 출원번호 확인 (예: 10-2026-0XXXXXX)
2. 출원증 다운로드 및 보관
3. 심사 진행 상황 주기적 확인 (특허로 → 마이페이지)

## 출원 후 주의사항

### 심사 과정
1. **출원공개**: 출원일로부터 18개월 후 자동 공개
2. **심사청구**: 출원과 동시에 심사 청구한 경우, 별도 절차 불필요
3. **심사 기간**: 심사청구 후 평균 10-14개월 소요
4. **거절이유통지**: 수령 시 2개월 이내 의견서 + 보정서 제출
5. **등록결정**: 등록 결정 후 3개월 이내 등록료 납부

### 의견서 대응 전략
거절이유통지를 받을 경우:

1. **신규성 거절**: 선행기술과의 차이점을 명확히 주장
   - 핵심 차별점: "프랜차이즈 네트워크 + 다중 플랫폼 + 차등 정산"의 3단 조합
   - 어떤 선행기술도 이 3가지를 하나의 시스템에서 통합하지 않음

2. **진보성 거절**: 기술적 시너지 효과 주장
   - 단순 결합이 아닌 유기적 통합으로 인한 새로운 기술적 효과
   - Firestore onSnapshot 기반 밀리초 단위 전파의 기술적 진보성
   - 외부 임베드 + 자체 커머스 하이브리드의 독창성

3. **발명의 단일성 거절**: 분할출원 대응
   - 심사관이 "2개 이상의 발명"으로 판단 시 분할출원 가능
   - 분할 후보: 시스템 발명(청구항1-6,12) / 방법 발명(청구항7-9) / 정산 발명(청구항10-11)
   - 분할 시 원출원일 유지 (출원일 이익)

- 라이브 커머스 MVP: Firestore 실시간 채팅 + 외부 임베드
- 프랜차이즈 네트워크: parentMallId/childMallIds 계층 구조 구현
- 정산 시스템: Cloud Functions 기반 자동 정산 구현
